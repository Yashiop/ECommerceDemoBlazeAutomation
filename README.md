E-Commerce DemoBlaze Automation Framework

About Project
This is a Selenium Automation Testing Framework developed for automating the DemoBlaze E-Commerce website using:
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)

Tech Stack
- Java
- Selenium
- TestNG
- Maven
- POM Design Pattern

Features Automated
- Login
- Product selection
- Add to cart
- Place order
- Checkout process

How To Run
1. Clone the project from GitHub.
2. Open in Eclipse or IntelliJ.
3. Run testng.xml or any test class from src/test/java.

Framework Highlights
- Page Object Model (POM)
- Config driven using properties files
- Reusable common methods
- Maven dependency management
- TestNG execution
